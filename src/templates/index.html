About the Model

This prediction tool is powered by a Machine Learning model trained on employee performance data. The model uses factors such as environment satisfaction, salary hikes, department, work-life balance, and experience to predict the likely performance rating.

The model was evaluated using XGBoost, Random Forest, and LightGBM, with the best performing algorithm selected (XGBoost) after hyperparameter tuning. Metrics such as Accuracy and F1-score were used to validate performance.

Predictions fall into one of the following categories:

  • Good – Meets expectations consistently.
  • Excellent – Often exceeds expectations.
  • Outstanding – Consistently exceeds expectations with exceptional results.

Disclaimer: Kindly note that this tool is for analysis, not official HR decision-making. Predictions are based on historical data trends and should not replace professional performance evaluations.
